# [L4D2] 允许电锯攻击中拾取 / Allow Chainsaw Pickup While Attacking

**插件名称 / Plugin Name:** `l4d2_chainsaw_canpickup`  
**版本 / Version:** 2026/7/18  
**作者 / Author:** qy087  
**适用游戏 / Game:** Left 4 Dead 2 (Only)

---

## 简介 / Description

**中文**  
玩过L4D2的都知道，拿起电锯一按左键，链条转起来的时候，你啥都捡不了——地上的枪、药包、投掷物，眼睁睁看着就是捡不起来，必须等电锯停下来才行。  
这个插件就是解决这个烦人问题的。装上之后，**电锯转着的时候也能正常捡东西**，不用再傻等锯子停转了。不影响电锯的伤害和使用，只是让你操作更顺手。

**English**  
You know that annoying thing in L4D2? When you hold a chainsaw and start swinging, you can't pick up anything—guns, medkits, throwables—all right there but you can't grab them until the saw stops spinning.  
This plugin fixes that. Once installed, **you can pick up items even while the chainsaw is attacking**. No more waiting for the saw to cool down. It doesn't change damage or anything else, just makes your life easier.

---

## 它有什么用？ / What's the benefit?

- ✅ 允许在电锯攻击动画中拾取任何物品（武器、投掷物、药包等）。  
- ✅ 不干扰其他武器的正常使用。  
- ✅ 轻量级实现，性能开销极小。

## 安装 / Installation

1. 将编译后的 `.smx` 文件放入 `addons/sourcemod/plugins/` 目录。  
2. 重新加载插件或重启服务器。  
3. （可选）若需要手动加载，控制台输入 `sm plugins load l4d2_chainsaw_canpickup`。
